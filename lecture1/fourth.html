Web citaci

Web citac je aplikacija koja je namenjena prikazu web stranica. Stranice se mogu dobaviti kako iz fajl sistema, tako i putem TCP/IP mreze. Protokol kojim komuniciraju web citac i web server je HTTP protokol. Danas su najpoznatiji sledeci web citaci:

  1. Mozzila Firefox
  2. Google Chrome
  3. Opera
  4. Postoje i drugi
    • Microsoft Edge
    • Brave

Obratite paznju da su svi nazivi web citaca ujedno i linkovi koji vode do njihovih stranica
